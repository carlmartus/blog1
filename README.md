# Martus.se blogging system
To be used on [martus.se](http://www.martus.se).

## Requirements
 * LuaJIT.

Following [Luarocks](https://luarocks.org/) modules:
 * TurboLua (`turbo`)
 * cjson (`lua-cjson`)
 * cmark-lua (`cmark`)
 * Lua File System (`luafilesystem`)
